Deploy in three steps

Download a template

Grab the HTML file from our downloads folder or clone the repo. Edit copy, colors, and images in any text editor.

Push to GitHub

Create a repository, add your files, and connect it to Cloudflare Pages. Set build command to none — output directory is root.

Point your domain

Add a CNAME record at your registrar. SSL provisions automatically. Your page is live worldwide on Cloudflare's CDN.
