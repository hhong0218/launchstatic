How to use a template

Pick a template

Open the detail page, preview the live demo, and download the HTML file when it matches your use case.

Edit copy and colors

Change product name, headlines, and CSS variables at the top of the file. No separate stylesheet to hunt down.

Deploy to any static host

Upload to Cloudflare Pages, Netlify, or GitHub Pages. Point your domain and you are live — no build command required.
